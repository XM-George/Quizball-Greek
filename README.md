# ⚽ QuizBall
Ένα διαδραστικό παιχνίδι quiz βασισμένο στο ποδόσφαιρο, όπου οι χρήστες μπορούν να δοκιμάσουν τις γνώσεις τους γύρω από ομάδες, παίκτες, διοργανώσεις και ιστορικές στιγμές του αθλήματος.  
Το project επικεντρώνεται στη δημιουργία ενός διασκεδαστικού football quiz με καθαρή δομή και εύχρηστο interface.

## 🚀 Χρήση <a name="usage"></a>
Ένα απλό football quiz.

- Ξεκίνα το quiz και απάντησε στις ερωτήσεις
- Κέρδισε πόντους για κάθε σωστή απάντηση
- Δες το τελικό σου score στο τέλος του παιχνιδιού
- Κάνε restart και προσπάθησε να πετύχεις καλύτερο αποτέλεσμα

## 📍 Η διαδικασία <a name="the-process"></a>
Αυτό το project δημιουργήθηκε γιατί ήθελα να δημιουργήσω ένα project βασισμένο στο ποδόσφαιρο, καθώς αποτελεί ένα από τα αγαπημένα μου αθλήματα.

Ξεκίνησα σχεδιάζοντας το interface της εφαρμογής με Java Swing. Δημιούργησα ένα απλό αλλά λειτουργικό GUI χρησιμοποιώντας frames, buttons, labels και panels ώστε ο χρήστης να μπορεί εύκολα να αλληλεπιδρά με το quiz.

Στη συνέχεια υλοποίησα το σύστημα ερωτήσεων και απαντήσεων. Οι απαντήσεις εμφανίζονται απλά χωρίς να ελέγχονται απο κάποιο σύστημα, πρέπει να ελέγχονται από τον ίδιο τον χρήστη.

Έπειτα πρόσθεσα το σύστημα βαθμολόγησης ώστε να υπολογίζονται οι σωστές απαντήσεις και να εμφανίζεται το τελικό αποτέλεσμα στο τέλος κάθε παιχνιδιού.

Αφού ολοκληρώθηκε η βασική λειτουργικότητα του παιχνιδιού, επικεντρώθηκα στη βελτίωση του user experience, διορθώνοντας bugs και βελτιώνοντας τη ροή μεταξύ των ερωτήσεων.


**Key learnings:**
- Δημιουργία GUI εφαρμογών με Java Swing
- Διαχείριση events και user input
- Οργάνωση multi-class εφαρμογών
- Υλοποίηση λογικής παιχνιδιού και συστήματος βαθμολόγησης
- Debugging και βελτίωση σταθερότητας εφαρμογών

## 💾 Εγκατάσταση <a name="installation"></a>

Ακολούθησε τα παρακάτω βήματα για να κάνεις clone και να τρέξεις το project locally. Υποστηρίζεται σε **Linux, macOS και Windows**.

### 1. Clone το repository

Βεβαιώσου ότι έχεις εγκατεστημένο το Git και τρέξε:

```bash
git clone https://github.com/XM-George/Quizball-Greek.git

cd Quizball-Greek

```

### 2. Άνοιξε το project
Μπορείς να ανοίξεις το project σε
- IntelliJ IDEA(συνιστάται)
- Eclipse
- οποιοδήποτε άλλο Java IDE

Βεβαιώσου ότι το **src** folder είναι ορισμένο ως source root αν χρειάζεται.

### 3. Τρέξε την εφαρμογή

#### Αν χρησιμοποιείς IDE:

Πήγαινε στο:

```bash

src/GUI/Main.java

```

και τρέξε απευθείας την **Main** class.

#### Αλλιώς αν χρησιμοποιείς το terminal

```bash

cd src
javac GUI/Main.java
java GUI.Main

```
